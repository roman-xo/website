# website
source code for my site, hosted at: https://rjtodd.com

## What's included/priorities

- static index under '/stuff/'
- built-in contact form with backend email delivery
- lightweight HTML/CSS — no frameworks, no fluff.
- optimized for speed

## Getting started

### To spin up a local copy:

```bash
git clone https://github.com/roman-xo/website.git
cd your-repo-name
python3 -m http.server 8000
```
Open any browser and navigate to:
http://localhost:8000
